testapi6-grpc

testapi6-grpc

Execute grpc command

Features
  • Create a gRPC server to mock data
  • Provide gRPC client to call to another

Read document details

How to use

Installation

 
// install via npm
npm install -g testapi6-grpc

// install via yarn
yarn global add testapi6-grpc

Use in yaml

 
- Require:
    root: path_to_this_modules
    modules:
      - testapi6-grpc/dist/index.js
- gRPCServer:
    packages:
      user:
        proto: /Users/doanthuanthanh/code/github/testapi6-grpc/src/server.proto
        services:
          RouteUser:
            GetUsers: {
              code: 1,
              data: [{name: 'thanh', age: 1}]
            }
- gRPC:
    title: Test call to a gRPC server
    proto: /Users/doanthuanthanh/code/github/testapi6-grpc/src/server.proto
    package: user
    service: RouteUser
    function: GetUsers
    arg: null
    timeout: 1000
    debug: details
    validate:
      - title: Check something
        func: length
        args:
          - ${$.response.data}
          - 1
